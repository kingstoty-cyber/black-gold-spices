# تطبيق ORA CAFÉ

تطبيق ويب متكامل لمقهى ORA CAFÉ يسمح للعملاء بتصفح القائمة وإضافة المنتجات إلى السلة وتقديم الطلبات.

## المميزات

1. **واجهة مستخدم عربية** متكاملة
2. **تصميم متجاوب** يعمل على جميع الأجهزة
3. **قائمة طعام كاملة** مع تصنيفات
4. **نظام سلة مشتريات** متكامل
5. **إدارة المفضلة**
6. **عرض العروض والتخفيضات**
7. **تتبع الطلبات السابقة**
8. **ملف شخصي للمستخدم**
9. **صفحة اتصال وعن المقهى**
10. **إعدادات قابلة للتخصيص**
